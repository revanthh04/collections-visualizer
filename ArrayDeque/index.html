Circular Buffer

ArrayDeque uses a resizable circular array. The head and tail indices wrap around using bitwise AND: (index + 1) & (elements.length - 1). Capacity is always a power of 2 to make this wrapping efficient.

Key Characteristics

  • All operations at both ends are O(1) amortized
  • Faster than LinkedList for Deque/Queue/Stack usage
  • Capacity doubles (power of 2) when full
  • Does NOT allow null elements
