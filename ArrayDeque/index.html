📦 Circular Array Backend

An ArrayDeque uses a contiguous array but treats it as a circle using two pointers:

  • Head Pointer: Tracks the first element. Decrements when adding to the front.
  • Tail Pointer: Tracks the next available slot at the end. Increments when adding to the back.
  • Bitwise Wrapping: Power-of-2 capacity lets it wrap around instantly using index & (length - 1) instead of modulo.

⚡ Faster than LinkedList & Stack

It is the undisputed king of Queues and Stacks in Java:

  • Cache Friendly: Contiguous memory means zero node allocation and great CPU cache locality.
  • No Shifting: Adding/removing from either end is O(1) because only pointers move.
  • Better than Stack (not synchronized/legacy) and better than LinkedList (no node heap allocation).

⚙️ Common Methods & Time Complexity

Method Description Origin Time Comp.
addFirst(E e) / push(E e) Inserts at the head. Traverses backwards in the array. Deque / Stack O(1)*
addLast(E e) / add(E e) / offer(E e) Inserts at the tail. Traverses forwards in the array. Deque / Queue O(1)*
removeFirst() / pop() / poll() Removes the head element. Advances head pointer. Deque / Stack / Queue O(1)
removeLast() Removes the tail element. Retreats tail pointer. Deque O(1)

* Amortized O(1). If the array fills up (head == tail), it creates a new double-sized array and copies elements linearly, taking O(n) temporarily.
