Internal Structure

ArrayList uses an internal Object[] elementData array. When elements are added beyond the current capacity, a new array is created with newCapacity = oldCapacity + (oldCapacity >> 1) — approximately 1.5× the old size. Elements are copied using System.arraycopy().

Key Characteristics

  • Random access: get(i) and set(i, e) are O(1)
  • Add at end: Amortized O(1) — occasionally O(n) when resizing
  • Add/Remove at index: O(n) — requires shifting elements
  • Default capacity: 10 (when first element is added)

When to Use

Use ArrayList when you need fast random access and mostly add/remove at the end. Avoid it for frequent insertions/removals in the middle — use LinkedList instead.
