Red-Black Tree Rules

  • Every node is either RED or BLACK
  • Root is always BLACK
  • No two consecutive RED nodes (red node cannot have a red parent)
  • Every path from root to null has the same number of BLACK nodes
  • Null nodes (NIL leaves) are considered BLACK

Self-Balancing

After insertion/deletion, the tree rebalances using rotations (left/right) and recoloring to maintain the rules above, ensuring O(log n) height.
